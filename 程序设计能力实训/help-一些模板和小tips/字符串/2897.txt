给定一个由若干英文单词组成的字符串，生成一个由各首字母组成的缩写词（acronym），其中的 the，a，an，of，for 及 and 被忽略。

Input
输入数据有若干行，每行上有一个字符串。字符串中分隔单词的字符为一个空格或一个连字符 (’-’)。单词中不包含连字符。字符串的长度不超过 100 个字符。

Output
对于每一个字符串，输出结果，每行最后有一个换行符。

Examples
input
United States of America
PEOPLE’S REPUBLIC OF CHINA
The Asia-Pacific Economic Cooperation
Organization of the Petroleum Exporting Countries
United Nation Educational Scientific and Culture Organization
Universal Serial Bus
output
USA
PRC
APEC
OPEC
UNESCO
USB

#include<stdio.h>
#include<string.h>
#include<ctype.h> 
int main()
{
 char s[100],t[20][100];
 int i,j,m,n;
 while(gets(s)!=NULL)
 {
  m=0;n=0;
  for(i=0;i<strlen(s);i++)
 {
	if(s[i]==' '||s[i]=='-'){m++;n=0;}
	else {t[m][n]=toupper(s[i]);n++;}
  }
  for(j=0;j<=m;j++)
    if (strcmp(t[j],"THE")!=0&&strcmp(t[j],"OF")!=0&&strcmp(t[j],"A")!=0&&strcmp(t[j],"AN")!=0&&strcmp(t[j],"AND")!=0&&strcmp(t[j],"FOR")!=0)
		  printf("%c",t[j][0]);
  printf("\n");
  memset(s,0,100);
  memset(t,0,2000);
 }
}
